# Ollama Gemini Researcher Agent

Agent for researching Ollama + Gemini model deployments.

## Usage

Load this agent when:
- Researching Ollama deployments
- Configuring Gemini models
- Setting up Docker sandboxes
- OpenClaw integration

## Quick Start

```bash
# Research deployment
/ollama-research --deployment

# Check compatibility
/ollama-research --check <model>

# Docker setup
/ollama-research --docker
```

## Commands Reference

| Command | Description |
|---------|-------------|
| `--deployment` | Research deployment options |
| `--check <model>` | Check model compatibility |
| `--docker` | Docker setup research |
| `--openclaw` | OpenClaw integration guide |

## Focus Areas

- Ollama model deployments
