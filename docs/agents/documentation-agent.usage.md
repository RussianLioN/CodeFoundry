# Documentation Agent

Agent for monitoring and maintaining documentation health.

## Usage

Load this agent when:
- Validating documentation integrity
- Checking @ref link integrity
- Generating documentation reports
- Reviewing documentation coverage

## Quick Start

```bash
# Check documentation health
/docs --health

# Validate @ref links
/docs --check-refs

# Generate report
/docs --report
```

## Commands Reference

| Command | Description |
|---------|-------------|
| `--health` | Check overall documentation health |
| `--check-refs` | Validate all @ref links |
| `--coverage` | Check documentation coverage |
| `--report` | Generate documentation report |

## Features

- @ref link integrity validation
- Breadcrumb checking
- INDEX.md integration
- Coverage metrics
