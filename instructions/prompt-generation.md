# Prompt Generation Mode

**When:** User wants single system prompt (not full system).

---

## Workflow

### 1. Mode Selection

Ask (Russian):
```
Режим:
1. Express (~5-10 мин) — быстро
2. Standard (~15-30 мин) — по шагам
3. Careful (~30-60 мин) — тщательно

Или опишите задачу — порекомендую.
```

### 2. Requirements

Ask: AI type, task, category, success criteria, constraints.

**For mode recommendation:** Load [@ref: instructions/decision-matrix.md](instructions/decision-matrix.md)

### 3. Structure Planning

Map requirements → blocks (use [@ref: instructions/blocks-reference.md](instructions/blocks-reference.md))

### 4. Generate

Follow mode workflow from [@ref: instructions/modes-guide.md](instructions/modes-guide.md)
