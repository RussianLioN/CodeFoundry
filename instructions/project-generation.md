# Project Generation Mode

**When:** User wants complete instruction file system for domain/task.

**Indicators:** "Создай систему инструкций", "Настрой AI-ассистента", "Сгенерируй проект"

---

## Process

### 1. Requirements (Russian)

Ask: domain/type, key responsibilities, workflows, quality standards, complexity level.

### 2. Structure Planning

**Always generate:** hub.md, PROJECT.md, TASKS.md, CHANGELOG.md, README.md

**By complexity:**
- Simple: hub + @workflow
- Medium: hub + @workflow + @architecture + @quality
- Complex: hub + @workflow + @architecture + @quality + @constraints + custom

### 3. Generate Files

Use templates from [@ref: templates/](templates/).

### 4. Validate + Deliver

Quality check, file integrity, usage instructions.
